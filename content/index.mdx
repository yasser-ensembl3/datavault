# DataVault Documentation

Welcome to DataVault - your research results, organized and accessible.

## What is DataVault?

DataVault is a frontend that displays research results from a Notion database. It helps you organize, review, and access your research findings from web searches, API data, datasets, and more.

## Quick Links

- [Setup Guide](/docs/setup) - Get started with DataVault
- [Notion Schema](/docs/notion-schema) - Configure your Notion database
- [API Reference](/docs/api-reference) - API endpoints documentation

## Features

| Feature | Description |
|---------|-------------|
| **Filtering** | Filter by type, status, or source |
| **Search** | Quick search by title |
| **Description** | Expandable descriptions for each result |
| **Tags** | Multi-tag support for organization |
| **Preview** | Embedded preview for Google Docs/Sheets |

## Research Types Supported

- **Web Search** - Results from Google, Bing, etc.
- **API Data** - JSON responses, API findings
- **Dataset** - CSV, databases, structured data
- **Analysis** - Charts, reports, insights
- **Article** - Blog posts, news articles
- **Paper** - Academic papers, research documents
- **Website** - Useful websites, tools
- **Insight** - Personal observations, conclusions
